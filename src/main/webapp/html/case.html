Application case
1
制式警车
杭州市公安局
助力G20峰会安保

2016年9月，我司联合杭州市公安局、电子科大公共安全技术研究中心、吉利汽车研发的第一代警用标准处警/巡逻车应用于G20峰会安保，得到了多方好评。

第一代警用标准处警/巡逻车配备有人脸车辆识别系统，一体化警示系统、自适应组网通信系统、HDS阵控、热源搜索成像装置、电子侦察装置、定向传声和声波驱散等装置，并通过触屏中央控制系统实现集成操作。因其功能的全面性与操作的智能化，在G20峰会期间，制式警车主要应用于机场区域、主会场及国宾主要活动场所警戒区域，并作为前导车和警戒区域巡逻以及重点路口卡点盘查。
